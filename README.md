# Gift Guide Page

**Preview:**
https://yash-sardhara-48-teststore-ltpz7f0s.myshopify.com/pages/gift-guide

## Sections

### Gift-Banner.liquid

Hero banner section displayed at the top of the Gift Guide page.

**Features**

* Custom heading and description
* CTA button
* Desktop & mobile image support
* Configurable through Theme Editor

### Gift-Grid.liquid

Product showcase section displayed below the banner.

**Features**

* Responsive image grid layout
* Product quick-buy hotspots on images
* Product information popup/modal
* Theme Editor configurable content and products
